The Azure DevOps project for this repo is **`CasaColinaCare.com`** (project ID `35f6c5fd-67f9-409d-9d79-20174be17108`, org `dev.azure.com/jairo`). Bug work items referenced in `tests/bugs.spec.ts` and `AGENTS.md` (e.g. #207036–#207043) live here — confirmed by looking up #207038 and matching its title to the repo's known bug list.

There is a **different, unrelated** ADO project named `CasaColinaDestination.com` — do not confuse the two when creating or querying work items for this repo.

See [[project-casa-colina]] for the repo's own context.
